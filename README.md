# Optimizing Business Insights & Performance Reporting
*A Data Warehousing & BI Project using Snowflake, SQL, dbt, and Power BI*

## Project Summary
This project builds a scalable data warehouse and reporting solution to support business performance analysis. It includes a multi-star schema on Snowflake, automated ELT pipelines using dbt, and interactive Power BI dashboards designed for executive-level insights.

## Key Contributions

* Designed a multi-star schema data warehouse in Snowflake using SQL and dimensional modeling

* Built fact/dimension models aligned with business KPIs and reporting needs

* Developed dbt pipelines for automated, version-controlled data transformations

* Created Power BI dashboards with KPIs and drill-through views for performance monitoring


## Tech Stack

* Warehouse: Snowflake

* ELT: dbt, Git

* Modeling: Star Schema, SQL

* Visualization: Power BI


## Outcome
Delivered a reliable analytics foundation and dashboards that enable data-driven decision-making across business teams.
